## ✨ Características

- 🟢 **Servidores soportados** — Vanilla, Paper, Forge, Fabric y Folia
- ☕ **Java automático** — Descarga e instala la versión de Java correcta para cada servidor
- 📁 **Gestor de archivos** — Accede a la carpeta del servidor desde la app
- 🖥️ **Consola integrada** — Monitorea la salida del servidor en tiempo real
- 🔔 **Actualizaciones automáticas** — La app te avisa cuando hay una nueva versión

## 🚀 Instalación

### Windows
Descarga el instalador `.exe` y ejecútalo. Crea accesos directos en el escritorio y menú inicio automáticamente.

### Linux (RPM — Fedora / RHEL / openSUSE)
```bash
sudo dnf install ./GetMineHub-X.X.X-Installer.rpm
```

### Linux (DEB — Ubuntu / Debian / Mint)
```bash
sudo apt install ./GetMineHub-X.X.X-Installer.deb
```
